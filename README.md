# JRA Paste Analyst MVP

JRAスマホ版の出馬表をコピーして貼り付けると、1レース分の予想材料を整理するローカルWebアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. JRA出馬表ページの内容を全選択してコピー
3. 左側の貼り付け欄に貼る
4. 「解析」を押す

## できること

- 出走馬ごとの近走評価
- 同じレースで走った今回出走馬同士の抽出
- 今回コースに対する同競馬場、同距離、前後200m、完全同条件の出走経験
- 一緒に走った馬のその後を確認するための候補表示
- 好走時馬体重レンジの表示
- 近走一覧の確認
- 印とメモのブラウザ内保存

## 注意

貼り付けテキストでは通過順と上がりがくっつく場合があります。MVPでは、着順・人気・距離・馬場・馬体重・勝ち馬名/着差を優先して解析します。
